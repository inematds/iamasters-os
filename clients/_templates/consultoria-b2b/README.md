# Template · Consultoría B2B

> Para consultores B2B (estrategia, ops, transformación) que sirven empresas medianas/grandes con engagement de 3-12 meses.

## Para quién

- Consultor solo o equipo de 2-5 (modelo high-ticket low-volume)
- Cada cliente: 30K-300K € de revenue por engagement
- Engagement típico: 3-12 meses
- Output principal: estrategia documentada + acompañamiento ejecución
- Tu reputación + tu network es el principal activo

## Diferencia con otros templates

| Aspecto | freelance-ia | agencia-marketing | formador-online | consultoria-b2b |
|---|---|---|---|---|
| Ticket promedio | 5-50K € | 2-15K €/mes | 200-2000 € | 30-300K € |
| Cliente | Pyme | Pyme/scaleup | Personas | Empresa mediana/grande |
| Decisión | Fundador | CMO/socio | Persona individual | Comité (CEO + 2-3) |
| Voice principal | Tu voice | Voice cliente | Tu voice | Voice consultor + adaptable corporate |
| Frecuencia output | Por proyecto | Mensual recurrente | Diario alto volumen | Quarterly + on-demand |

## Qué viene preconfigurado

- **Voice profile**: formal-autoridad, equilibrado, low-warmth high-precision (mayoritariamente A con B para thought leadership)
- **Positioning**: especialista vertical o método propio, anti-Big4 si aplica
- **ICP**: empresa mediana 50-500 empleados con problema estratégico claro
- **Estructura**: orientada a deliverables formales (decks, reports, frameworks)

## Skills prioritarias

1. `marketing-positioning` (clave para diferenciar de Big4 y boutiques)
2. `marketing-icp` (ICP muy específico = ticket alto)
3. `marketing-copywriting` (LinkedIn thought leadership semanal)
4. `tool-output-verifier` (formal A obligatorio en propuestas)
5. (futuras) `operations-meeting-notes` para reuniones ejecutivas
